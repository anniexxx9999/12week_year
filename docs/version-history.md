# 版本历史

本文档记录项目的所有版本发布历史。

## 版本命名规则

我们使用语义化版本号（Semantic Versioning）：
- 主版本号：不兼容的 API 修改
- 次版本号：向下兼容的功能性新增
- 修订号：向下兼容的问题修正
